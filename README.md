# ShaderConverter
Convert shaders to gdshaders from godot 3.+ to godot 4.+

### How to

1. Paste all *.shader (Godot 3.+) files to your project.
2. Go to Menu -> Project -> Tools -> Convert shaders to gdshaders
3. Use your shaders as needed
